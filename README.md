# COLLABRIX – Full Stack Project Management System

A modern full-stack project management application designed for teams to manage projects, tasks, workflows, and collaboration efficiently.

---

# Features

- User Authentication & Authorization
- Create & Manage Projects
- Board Task Management
- Drag & Drop Task Workflow
- Task Status Tracking
- Task Priority Management
- Create / Update / Delete Tasks
- Timeline View
- Table View
- List View
- Responsive UI
- Dark Mode Support
- Team Workspace Management


---

# Tech Stack

## Frontend
- Next.js
- React.js
- TypeScript
- Tailwind CSS

## Backend
- Node.js
- Express.js
- Prisma ORM

## Database
- PostgreSQL
